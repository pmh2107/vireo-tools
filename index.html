Job Description Evaluator

Built with ADHD in mind, useful for anyone whose brain doesn't love chaos.

This filter evaluates job descriptions across four dimensions that affect how neurodivergent brains interact with work environments. No diagnosis required — these signals matter for anyone whose brain doesn't love chaos.

1. Cognition — does the work reward how your brain processes information?
Green signals
  • Pattern recognition / synthesis valued
  • Complex, ambiguous problem spaces
  • Results matter more than performance style
  • "Build from the ground up" / "greenfield"
  • Independent thinking / "constructive challenge"
  • Deep expertise development possible
Red flags
  • Rigid procedural execution
  • "Strict adherence" / "process compliance"
  • Repetitive operational work
  • "Dashboard" / "reporting maintenance"
  • Output volume over insight quality
2. Energy regulation — does the environment support sustainable functioning?
Green signals
  • Remote / flexible / async options
  • Flexible hours
  • Protected focus / deep work time
  • Quiet spaces / sensory considerations
  • Reasonable meeting load
  • Scheduled breaks / sustainable pace
Red flags
  • "Always on" / immediate response culture
  • Open plan / hotdesking environment
  • Excessive meetings / "alignment" culture
  • Heavy context switching
  • Constant social performance required
  • Sensory-hostile environment signals
3. Ambiguity — what kind of ambiguity is present?
Green signals
  • Ambiguous problem spaces (good)
  • Complex, undefined challenges
  • "Synthesize" / "make sense of"
  • Proactive / strategic framing
Red flags
  • Role / expectation ambiguity (bad)
  • Hidden rules / unwritten expectations
  • Shifting priorities without rationale
  • "Role will evolve" / "responsibilities may shift"
  • No feedback structure mentioned
4. Organizational psychology — does the culture reward competence and intellectual honesty?
Green signals
  • Psychological safety language
  • Written / explicit communication culture
  • Regular structured feedback / check-ins
  • Mentorship / support culture
  • Visible impact / work affects decisions
  • "Different perspectives" / "independent thinkers"
  • Neurodiversity / DEI awareness signals
Red flags
  • "Culture fit" without specifics
  • Heavy political navigation required
  • "Manage up" / "influence without authority"
  • Insights with no visible influence
  • Micromanagement signals
  • Performance theater / optics over outcomes
  • Hidden curriculum / unwritten rules culture
